1、功能说明

    /* 简单描述工程功能 */
        这个例程配置并演示CAN1和CAN2在正常工作模式下相互收发CAN报文


2、使用环境

    /* 硬件环境：工程对应的开发硬件平台 */
        开发板：N32WB45xL_EVB V1.1
        CAN收发器：CAN1和CAN2都外接了CAN收发器


3、使用说明
    
    /* 描述相关模块配置方法；例如:时钟，I/O等 */
        SystemClock：144MHz
        GPIO：USART（TX - PA9，RX - PA10），CAN1（RX - PD0，TX - PD1），CAN2（RX - PB5，TX - PB6）
        CAN1/2：波特率500K，正常工作模式

    /* 描述Demo的测试步骤和现象 */
        1.编译后下载程序复位运行；
        2.查看串口打印信息，当查看到CAN1/2收发都正常的打印信息后表面程序运行正常；
